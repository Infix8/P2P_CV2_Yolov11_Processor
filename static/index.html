Worker nodes creator

Run worker processes on this device (or any device on the same network). Each worker advertises itself and the orchestrator discovers it automatically. Use a separate terminal per worker.

Install (worker-only, lighter):

pip install -r requirements-worker.txt

Start workers (one command per terminal):

python -m pipeline.worker_server 9001
python -m pipeline.worker_server 9002
python -m pipeline.worker_server 9003
python -m pipeline.worker_server 9004

Any device can run both the orchestrator (this app) and workers. Open the Orchestrator start tab to upload/camera and see results; use this tab as a reference to start worker nodes.
